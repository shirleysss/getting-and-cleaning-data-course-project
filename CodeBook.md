# Code Book
This cood book describes the contents of the data set.

## Identifiers
- `subject`: the ID of the test subject
- `activity`: the type of activity that they were performing when the measurements were taken
..* WALKING
..* WALKING_UPSTAIRS
..* WALKING_DOWNSTAIRS
..* SITTING
..* STANDING
..* LAYING
